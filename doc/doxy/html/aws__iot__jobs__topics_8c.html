int aws_iot_jobs_get_api_topic ( char *  buffer,
size_t  bufferSize,
AwsIotJobExecutionTopicType  topicType,
AwsIotJobExecutionTopicReplyType  replyType,
const char *  thingName,
const char *  jobId 
)

Get the topic matching the provided details and put into the provided buffer.

If the buffer is not large enough to store the full topic the topic will be truncated to fit, with the last character always being a null terminator.

Parameters
bufferthe buffer to put the results into
bufferSizethe size of the buffer
topicTypethe type of the topic
replyTypethe reply type of the topic
thingNamethe name of the thing in the topic
jobIdthe name of the job id in the topic
Returns
the number of characters in the topic excluding the null terminator. A return value of bufferSize or more means that the topic string was truncated.
